AppCmd

The fastest way to switch application windows on macOS :)

Install via Brew GitHub

Switching & Cycling

Hold the Right Command key and press a letter to switch to apps starting with that letter. Tap the key again to cycle through multiple matches.

Custom Assignments

Create static mappings to ensure specific apps always respond to specific keys. Focus an app and press Right Command + Option + [Letter] to assign it.

Visual Cheat Sheet

Hold Right Command to reveal a grid of all active assignments. The delay is fully configurable in the settings window.

Showcase

Quick Installation

brew tap clearnote01/tap brew install appcmd brew services start appcmd

AppCmd requires Accessibility and Input Monitoring permissions to detect the Right Command key and manage window switching.
